# 🚀 OmniDownloader v2.5.0
**منصة احترافية لتحميل الوسائط متعددة المنصات**

[English Version 🌐](./README.md)

---

## 🌐 نظرة عامة

OmniDownloader هو حل متميز وعالي الأداء لتحميل الوسائط، تم بناؤه باستخدام **Tauri v2** و **React** و **Rust**. يوفر واجهة موحدة وأنيقة للبحث والتحليل وتحميل الوسائط عالية الجودة من YouTube و Telegram وأكثر من 1000 موقع آخر. يتبع المشروع مبادئ **التصميم الذري (Atomic Design)** ويعتمد على منطق برمجيات (Hooks) قوي لضمان الكفاءة والقابلية للتوسع.

---

## ✨ الميزات الرئيسية

- **📥 معالجة الدفعات**: دعم تحميل روابط متعددة في وقت واحد.
- **⚡ استخراج ذكي**: دعم متخصص لروابط Telegram وأكثر من 1000 موقع عبر `yt-dlp`.
- **🧩 ملحق المتصفح**: ملحق متكامل لاستخراج الروابط مباشرة من المتصفح.
- **🎨 واجهة مستخدم متميزة**: واجهة احترافية بالوضع الليلي محسنة باستخدام Tailwind CSS.
- **🏗️ بنية ذرية**: هيكل مكونات منظم للغاية (Atoms, Molecules, Organisms).
- **📋 سجلات فورية**: تتبع مباشر لتقدم التحميل وأنشطة النظام.

---

## 🛠️ التقنيات المستخدمة

- **الواجهة الأمامية**: React 19, TypeScript, Vite, Tailwind CSS, Radix UI.
- **الواجهة الخلفية**: Rust (Tauri v2), ملفات ثنائية جانبية (`yt-dlp`, `wget`).
- **محرك الوسائط**: FFmpeg (مدمج لدمج الملفات بجودة عالية).
- **إدارة الحالة**: Custom Hooks و React Context.

---

## 🚀 ابدأ الآن

### المتطلبات الأساسية
- **Node.js**: (يفضل استخدام pnpm)
- **Rust Toolchain**: مطلوب لتجميع Tauri.
- **FFmpeg**: مدمج في مجلد المشروع الرئيسي.

### التثبيت
1. قم بنسخ المستودع:
   ```bash
   git clone https://github.com/ahmadabdelbary2001/OmniDownloader.git
   ```
2. تثبيت الاعتمادات:
   ```bash
   pnpm install
   ```
3. التشغيل في وضع التطوير:
   ```bash
   pnpm tauri dev
   ```

## 📂 هيكل المشروع

- **`src/`**: منطق الواجهة الأمامية (React).
  - **`components/`**: المكونات مقسمة حسب التصميم الذري.
  - **`hooks/`**: المنطق الأساسي (محرك التحميل، مدير العمليات، محلل الروابط).
- **`src-tauri/`**: ملفات وتكوينات Rust.
  - **`bin/`**: الملفات الثنائية الجانبية لـ Windows.
- **`extension/`**: الكود المصدري لملحق المتصفح.
- **`ffmpeg.exe`**: ملف ثنائي مدمج لمعالجة الوسائط.
